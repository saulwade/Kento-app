# Kento

Kento is an operations management platform for restaurants.

It helps restaurant owners and managers control:
- inventory
- recipe-based ingredient consumption
- food cost
- staff operational compliance

## Stack
- Next.js
- React
- TailwindCSS
- Clerk
- Convex

## MVP modules
- Dashboard
- Inventory
- Menu & Recipes
- Food Cost
- Staff

## MVP goal
Build the cleanest possible first version that proves operational value for restaurants.
