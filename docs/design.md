# Kento Design Direction

Kento should feel:
- premium
- minimal
- calm
- modern
- operationally focused

## Visual style
Inspired by modern SaaS landing pages and dashboards with:
- light backgrounds
- floating cards
- lots of whitespace
- subtle shadows
- rounded corners
- elegant typography
- restrained color palette

## Color tokens

- gray-100: #f8f9fc
- gray-200: #f1f3f9
- gray-300: #dee3ed
- gray-400: #c2c9d6
- gray-500: #8f96a3
- gray-600: #5e636e
- gray-700: #2f3237
- gray-800: #1d1e20
- gray-900: #111213

## Shadows

Default:
0 2px 6px -1px rgba(0, 0, 0, 0.16), 0 1px 4px -1px rgba(0, 0, 0, 0.04)

Active:
0 0 8px -2px rgba(0, 0, 0, 0.1), 0 6px 20px -3px rgba(0, 0, 0, 0.2)

## Typography
Use a clean sans-serif style similar to Inter or Work Sans.

## UI principles
- Keep each page focused
- Avoid overcrowding
- Keep actions obvious
- Prioritize readability
- Build interfaces that feel premium but easy to use
