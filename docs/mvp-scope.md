# Kento MVP Scope

## Build now
- authentication
- workspace structure
- dashboard shell
- ingredients CRUD
- menu items CRUD
- recipes CRUD
- food cost calculation
- checklists CRUD
- checklist completion UI

## Build later
- POS integrations
- purchase order automation
- multi-location support
- advanced analytics
- supplier portal
- payroll
- scheduling
- AI forecasting

## MVP rule
If a feature does not directly help validate Kento's operational value for restaurants, do not build it yet.
