# Kento User Flows

## Flow 1: First-time setup
1. User signs up
2. User creates restaurant workspace
3. User adds first ingredients
4. User creates first menu item
5. User attaches recipe
6. User views dashboard

## Flow 2: Inventory setup
1. Open Inventory
2. Add ingredient
3. Set unit and cost
4. Set current stock
5. Save ingredient

## Flow 3: Recipe creation
1. Open Menu & Recipes
2. Create menu item
3. Set selling price
4. Add recipe ingredients
5. Save recipe
6. Food cost updates automatically

## Flow 4: Staff checklist
1. Manager opens Staff
2. Creates checklist by role or shift
3. Employee opens checklist
4. Marks tasks complete
5. Manager sees completion status

## Flow 5: Dashboard review
1. Owner opens dashboard
2. Reviews alerts
3. Checks food cost summary
4. Reviews staff completion rate
5. Takes action if needed
